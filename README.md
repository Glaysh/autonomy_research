# Autonomy Research

This repository contains datasets and analytical notebooks for a research project on learner autonomy in Russian as a Foreign Language (RFL) teaching.

## Overview

The project investigates how teachers conceptualize learner autonomy, how different dimensions of autonomy are represented in their responses, and which pedagogical strategies are associated with autonomy-supportive teaching.

The repository includes both quantitative and qualitative materials:
- survey data
- interview data
- notebooks for statistical and interpretive analysis
- visualizations of factor structures and thematic patterns

## Contents

### Datasets
- Опрос_автономия.xlsx - survey data
- Copy of Автономия Интервью.xlsx - interview data

### Jupyter notebooks
- Survey_Autonomy_Analysis.ipynb - quantitative analysis of survey responses
- Interview_Autonomy_Analysis.ipynb - qualitative and mixed-method analysis of interview data

### Additional files
- README.md - repository description
- .gitignore - ignored files configuration

## Methods used

The analysis includes:
- descriptive statistics
- comparison across teaching formats
- factor analysis
- coding of open-ended responses
- lexical-semantic marker analysis
- visualization of factor loadings and thematic groupings

## Purpose

This repository was created to organize the analytical workflow, preserve datasets and notebooks, and support reproducibility of the research process.

## Language

The data and many variable labels are in Russian, as they reflect the original language of the study materials.
